# CareConnect Scripts

Utility scripts for CareConnect platform.

## Available Scripts

### 1. Security Check (`security-check.js`)
Validates security configuration and checks for sensitive files.

```bash
npm run security-check
```

### 2. Voice Startup (`voice-startup.js`)
Auto-starts CareConnect with voice control enabled.

```bash
npm run voice-startup
```

## Usage

All scripts can be run from the root directory using npm scripts defined in `package.json`.
