# Smart Home - Node.js Version

## Installation & Setup

1. Install Node.js dependencies:
```bash
npm install
```

2. Start the server:
```bash
npm start
```

3. Access the application:
- Open browser: `http://localhost:3000`
- Login credentials:
  - Username: `Admin`
  - Password: `admin`

## Features

- Same functionality as PHP version
- No database required (hardcoded authentication)
- Firebase real-time LED control
- Clean, modern interface
- Session management

## File Structure

- `server.js` - Main Express server
- `views/` - HTML templates
- `public/` - Static files (CSS, JS)
- `package.json` - Dependencies
